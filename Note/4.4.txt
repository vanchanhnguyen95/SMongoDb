Bài 4.4: Publishing ports
Run docker với ports mới ra bên ngoài (port public ra bên ngoài)
(ví dụ bên ngoài đã có cổng 4200, mình phải chuyển sang port khác khi public ra bên ngoài)
docker run -d -p 4201:4200 --name angular-app lemonnguyen/angular-app:v2

Config lại file dockerfile
FROM node:18.15.0-alpine3.17
RUN npm install -g @angular/cli@15.2.4
RUN addgroup app && adduser -S -G app app
USER app
WORKDIR /app
COPY package.json .
RUN npm install
COPY . .
ENV API_PATH =http://10.0.10.146:9200
EXPOSE 4201

# shell form
# CMD npm start
# Exec form: nhanh hơn, thuận tiện hơn
# CMD [ "npm" , "start" ]
# ENTRYPOINT [ "npm" , "start" ]
ENTRYPOINT [ "ng" ,"serve" ,"--host", "0.0.0.0", "--disable-host-check" ]

build lại
docker build -t lemonnguyen/angular-app:v2 .

- remove những container đang chạy
docker container stop $(docker container ls -aq) && docker system prune

-- Chạy lại docker
docker run -dp 4201:4200 --name angular-app lemonnguyen/angular-app:v2

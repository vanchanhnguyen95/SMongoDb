Bài 6.7: Reducing Image Size
**Cách giảm thiểu image size khi build docker
1. Vào thư mục backend
- cài thư viện rimraf: giúp remove thư mục dist sạch sẽ, tối ưu nhất
2. Vào thư mục backend
- thêm file Dockerfile.prod
# Step 1: Build
FROM node:14.16.1-alpine3.13 AS build-stage
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

# Step 2: Production
FROM nginx:1.12-alpine
COPY --from=build-stage /app/dist /usr/share/nginx/html
EXPOSE 80
ENTRYPOINT [ "nginx", "-g", "daemon off;" ]
